Intro to Northcoders

Northcoders is an intense, full-time coding bootcamp delivering over 500 hours of coding practice. The course can be broken into 4 clear sections:

Fundamentals

A deep dive into various programming concepts such as closure, recursion, higher-order functions and asynchronous programming. THe material was delivered through daily lectures followed by practice time with a carefully curated series of katas and projects. Throughout the entire course there was a strong emphasis on test driven development and pair programming.

Back-end

The back-end section of the bootcamp introduced us to creating servers using HTTPS createserver before moving on to express.js to develop more succinct models. Using the MVC model we created exciting and RESTful API's incorporating PSQL databases and hosting on Heroku using github workflow/actions to ensure continuous integration. A link to my final back-end project is available from the projects section
